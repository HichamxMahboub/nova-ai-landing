# Nova AI Landing Page

> **Status:** frontend concept and interaction study.

A responsive SaaS landing page built to practice product storytelling, component composition, motion, and polished interface details. It is a frontend demonstration rather than a live AI product.

## Highlights

- Clear hero message and call-to-action flow.
- Responsive bento-style feature layout.
- Reusable React components.
- Framer Motion entrance and interaction effects.
- Tailwind CSS design tokens and responsive styling.
- Lightweight Vite development and production builds.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Structure

```text
src/components/Hero.tsx          Hero and waitlist presentation
src/components/FeatureGrid.tsx   Feature showcase
src/App.tsx                      Page composition
src/index.css                    Global styles and visual system
```

## Local Setup

```bash
npm install
npm run dev
```

Validate the production build:

```bash
npm run build
npm run preview
```

## Scope

The waitlist and AI-related copy are interface concepts. No production AI service, user account system, or persistent backend is included.

## Portfolio Value

This project shows frontend execution: translating a product concept into a responsive, animated landing page with a clear hierarchy and reusable components.
