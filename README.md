# Nova AI Landing Page

Premium AI startup landing page built with React, Tailwind CSS, and Framer Motion.

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Structure

- `src/components/Hero.tsx` — Hero section with staggered animations and waitlist form
- `src/components/FeatureGrid.tsx` — Bento grid feature showcase
- `src/App.tsx` — Page composition and background glow layers
- `src/index.css` — Tailwind base styles
